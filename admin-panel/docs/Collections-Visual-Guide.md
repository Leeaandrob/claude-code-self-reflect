## Color Scheme

### Status Badges
- **Green**: Active collection (bg-green-500)
- **Red**: Error/inactive (bg-destructive)

### Visual Hierarchy
- **Primary**: Collection names (text-lg, font-semibold)
- **Secondary**: Metric labels (text-sm, text-muted-foreground)
- **Tertiary**: Values (font-semibold)

### Interactive States
- **Hover**: Shadow increase (hover:shadow-md)
- **Selected**: Blue ring (ring-2 ring-primary)
- **Focus**: Visible focus ring (focus-visible:ring-2)

## Icons Used

- **Database**: Collection card icon (lucide-react)
- **Search**: Search input icon
- **ChevronDown**: Sort descending indicator
- **ChevronUp**: Sort ascending indicator
- **RefreshCw**: Manual refresh button
